1. Назначение сайта
Сайт представляет собой платформу для публикации статей (постов) с возможностью регистрации пользователей, комментирования и взаимодействия между авторами.
Основные цели:

Создание сообщества для обмена знаниями.

Удобная публикация и чтение материалов.

Персонализация пользовательского опыта (аватарки, профили).

2. Функционал
Для пользователей:
Регистрация/авторизация через логин и пароль.

Создание/редактирование постов с заголовком и текстом.

Комментирование постов других пользователей.

Профиль пользователя с аватаркой, списком постов и активностью.

Просмотр ленты с пагинацией (разбивкой на страницы).

Для администраторов:
Управление контентом (модерация постов/комментариев).

3. Используемые технологии
Backend:

Язык: Python 3.

Фреймворк: Flask.

База данных: SQLite (с миграциями через Flask-Migrate и Alembic).

ORM: SQLAlchemy.

Аутентификация: Flask-Login.

Загрузка файлов: Flask-WTF.

Frontend:

Шаблонизация: Jinja2.

Стили: CSS (возможно с использованием Bootstrap).

Интерактивность: JavaScript (опционально).

Безопасность:

Защита от CSRF-атак (Flask-WTF).

Хеширование паролей (Werkzeug).

4. Структура базы данных
Таблица User:

id (Primary Key)

username (уникальный логин)

password_hash (хэш пароля)

avatar (путь к аватарке)

Таблица Post:

id (Primary Key)

title (заголовок)

content (текст)

created_at (дата создания)

author_id (Foreign Key → User.id)

Таблица Comment (опционально):

id (Primary Key)

text (текст комментария)

post_id (Foreign Key → Post.id)

user_id (Foreign Key → User.id)
